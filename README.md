Google Gemini Clone

A modern **Google Gemini AI Clone** built with **React.js** that allows users to interact with Google's Gemini AI model through the **Google Generative AI API**. The application provides a clean, responsive, and intuitive chat interface for generating AI-powered responses in real time.

---

## 🚀 Features

- 🤖 AI-powered chatbot using Google Gemini API
- 💬 Real-time conversation interface
- ⚡ Fast and responsive React application
- 🎨 Modern and user-friendly UI
- 📱 Fully responsive design
- 📝 Chat history during the session
- 🔄 Loading animation while generating responses
- ⚠️ Error handling for failed API requests
- 🌙 Clean and minimal interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS
- HTML

### API
- Google Gemini API (Google Generative AI)
